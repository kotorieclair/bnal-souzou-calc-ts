# bnal-souzou-calc

文豪とアルケミスト（文アル）の文豪と装像から攻撃・防御・回避の増減値をシミュレートするツール。  
文豪個別のステータス入力にも対応、攻撃・防御・回避の最終的な数値も算出可能。  
4人分入力可能なので、会派編成にも同一文豪での装像比較にも使えます！
